About Me:

HI! I'm Tomasz, a beginner self-taught front-end developer. Currently working as a CNC Programmer with almost 10 years of experience in this field. I always had a passion and a knack for automation, making things simpler and more user friendly. I've turned my attention front-end and web development as it is evolving, a fast-developing sector which gives a lot of opportunities to learn and grow! 

For the past year I've been learning from online courses and building small projects to hone my skills and knowledge. You can see some of them in the Projects section.

In my free time, I like to be active! I spend a lot of time working out, hiking and attending dance classes. I'm also very dedicated to self-improvement, always looking to make myself a little bit better than a day before, which also translates to my work-life.

Pig-Game

A simple game for 2 players where the goal is to collect 100points by rolling the dice. Project built do exercise DOM manipulation and incorporating simple logics.

Technologies used: HTML5, CSS3, JS.


Live version: https://pig-game-tp.netlify.app/

Github repo: https://github.com/Pakul91/Pig-Game

Project built following the tutorial made by Jonas Schmedtmann.

Bankist

A small online banking simulator where the user can log between two accounts, transfer money between them, check balance or request a loan. Project built to practise using arrays in real-life apps, work with internationalization timezones and counters.

Technologies used: HTML5, CSS3, JS, Intl API.


Live version: https://bankist-tp.netlify.app/

Github repo: https://github.com/Pakul91/bankist-tp

Project built following the tutorial made by Jonas Schmedtmann.

Bankist

A dynamic landing website for the fictional bank. The project was built to practice advanced DOM manipulation in vanilla JavaScript and working with Intersection Observer API. Creating features like menu fading, tabbed components, image lazy loading or revealing images on scroll. HTML and CSS were provided.

Technologies used: HTML5, CSS3, JS, Intersection Observer API.


Live version: https://bankist-website-tipi.netlify.app/

Github repo: https://github.com/Pakul91/bankist-website-tipi

Project built following the tutorial made by Jonas Schmedtmann.

MAPTY

Simple application to log your running and cycling activities with handy pins to show them on the map. Application built to practise working with classes, geolocation and external APIs.

Technologies used: HTML5, CSS3, JS, Geolocation API, Leaflet API.


Live version: https://mapty-tp.netlify.app/

Github repo: https://github.com/Pakul91/mapty

Project built following the tutorial made by Jonas Schmedtmann.

Forkify

A simple app to search for cooking recipes using an API database with over 100 000 recipes. Additionally, the user can add bookmarks to store favourite recipes and create custom-made ones and store them in the browser memory. There is no external database. Project built to practise working with modules, classes, asynchronus JS, external APIs and NPM

Technologies used: HTML5, CSS3, JS, Forkify API, NPM and Parcel


Live version: https://forkify-tomasz.netlify.app

Github repo: https://github.com/Pakul91/forkify-course-video

Project built following the tutorial made by Jonas Schmedtmann.

Reader-fo_reddit

Reader for Reddit is a simple web app allowing users to search for interesting topics or communities on Reddit, scroll through the content, see a detailed view of posts, and check comments. App built to practice using React functional components in conjunction with React Router and Redux Toolkit.

Technologies used: HTML5, CSS3, JS, React 17, Redux, React-router, Reddit JSON API


Live version: https://reader-for-reddit-tipi.netlify.app/

Github repo: https://github.com/Pakul91/reader-for-reddit

Project built as a portfolio project from codecademy.com.
